Mock Webiste

Lorcana Legends Expo is a multi-page event website inspired by Disney’s Lorcana TCG. Designed and developed from concept to completion, it combines cinematic fantasy visuals with clean, accessible front-end design. Each page - Home, Guests, Gallery, Schedule, Register, and FAQ - uses consistent color, typography, and animation to create an immersive collectible-card atmosphere. Built entirely with HTML5 and CSS3 the project demonstrates responsive design, visual hierarchy, and cohesive brand storytelling through a professional, themed event experience.
